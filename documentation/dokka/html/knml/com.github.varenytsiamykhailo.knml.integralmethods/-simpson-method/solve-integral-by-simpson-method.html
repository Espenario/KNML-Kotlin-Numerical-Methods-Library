Simpson method implementation.

The Simpson method is a method for the numerical integration of a function of one variable. The essence of Simpson's method is to approximate the integrand on a segment by an interpolation polynomial of the second degree, that is, to approximate the function graph on a segment by a parabola.

Asymptotic complexity: O(n^2 * T(integralFunction)), where T - asymptotic complexity of the integral function.

See Also:https://en.wikipedia.org/wiki/Simpson%27s_rule, https://ru.wikipedia.org/wiki/Формула_Симпсона

Return

This method returns approximate integral solution of the input integral function which is wrapped into DoubleResultWithStatus object. This object also contains integral solution of Double value, successful flag, error-exception object if unsuccess, and solution object if needed.

Parameters

intervalStart

is the start of integration interval.

intervalEnd

is the end of integration interval.

eps

is the input required precision of the result. The user can use, for example, 'eps = 0.001' if he need quickly solution with low error. If the user does not pass their required precision, then will be used default machine precision as the most accurate precision.

formSolution

is the flag, that says that the method need to form a solution object with the text of a detailed solution.

integralFunction

is the integral function, which will be integrated.
